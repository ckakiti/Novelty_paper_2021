# Data structure
Each .mat file is named based on the following file structure:

```
.
+-- group_name
|   +-- animal1_name
|       +-- session1_name
|           -- DLC_label.mat
|       +-- session2_name
|       +-- session3_name
|   +-- animal2_name
|   +-- animal3_name
|   -- bout.mat
|   -- bout_multi.mat
-- akiti_miceID_211222.xlsx
```
